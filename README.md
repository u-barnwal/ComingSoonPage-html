# Coming Soon Page
A website template built with HTML & CSS to quickly create a good-looking responsive single-page message page for your website.  

## Use Cases
- Coming Soon Page
- Error Pages (404, 403, etc.)
- Message Page

## Features
- Easy to modify
- Lightweight
- 100% Responsive 
- Social Media Links
